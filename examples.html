Examples

Basic

Real-world workflows easily grow to tens, maybe hundreds of operators. Editing them manually in canvas-based editors quickly become tedious and unwieldy. ADAMS automatically arranges workflows in a very compact way and uses control operators to direct the data flow.

Visualization

Drop in and pull out operators, play with parameters and settings, quickly write and plug-in new operators. ADAMS allows you to focus on creating and finetuning workflows.

Advanced

ADAMS contains an extensive library of operators, and a plug-in architecture to easily add new ones. Write a single java class, drop it in a folder, done!
